Supabase integration:
1. Create a Supabase project and add a videos table with columns: id, title, src, thumb, used, featured, featured_label, created_at.

Cloudflare R2 setup:
1. Create a bucket in Cloudflare R2 and enable public access.
2. Upload videos. Your public URL format: https://pub-[id].r2.dev/[filename]
3. Use those URLs as src and thumb in the video entries above.
